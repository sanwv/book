服务器配置

添加git用户,只可用git-shell

useradd -d /git -s /usr/bin/git-shell git

创建家目录和.ssh目录

mkdir -p /git/.ssh

把允许操作的用户的公钥保存到authorized_keys中

touch /git/.ssh/authorized_keys
chmod 0600 /git/.ssh/authorized_keys

用户

chown -R git:git /git
